# 『無響室の対話』連載計画

## 投稿プラットフォーム（3サイト同時投稿）

全サイトに同じ作品を同時投稿する。各サイトの規約上、同時掲載は許可されている（コンテスト応募時は個別に要項を確認すること）。各サイトのプロフィールおよび作品ページに「他サイトでも掲載中」と記載する。

### カクヨム（メインプラットフォーム）

- **URL：** https://kakuyomu.jp/
- **選定理由：** SFジャンル専用カテゴリ・ランキングがあり、SF読者が集まっている。KADOKAWA運営のため書籍化ルートが明確。年1回の「カクヨムWeb小説コンテスト」にSF部門あり。PV報酬制度（約0.12円/PV）もある
- **狙う読者層：** SF・文芸ファン、Web小説の習慣的な読者
- **活用する機能：** フォロー通知、ランキング、コンテスト応募、PV報酬

### Tales（テイルズ）

- **URL：** https://tales.note.com/
- **選定理由：** noteの姉妹サービスとして2025年4月にオープンした物語専用プラットフォーム。25のメディアパートナーと連携しており、書籍化・漫画化・映像化のチャンスがある。noteアカウントと連携可能。小説に特化した目次機能・連載管理がnoteより優れている
- **狙う読者層：** noteユーザーからの流入、エンタメ原作を探しているメディア関係者
- **活用する機能：** 目次機能、創作大賞への応募、メディア連携

### note

- **URL：** https://note.com/
- **選定理由：** エッセイ・ノンフィクション読者が厚く、「AIと社会」というテーマに関心のある一般読者にリーチできる。小説の連載機能としては弱いが、作品の周辺コンテンツ（執筆の裏話、テーマに関するエッセイなど）を発信する母艦として活用する
- **狙う読者層：** AI・テクノロジーに関心のある一般層、クリエイター層
- **活用する機能：** マガジン機能、ハッシュタグ、エッセイとの併用

## 1回あたりの文字数

3,000〜4,000文字を目安とする。スマホで「5〜10分で読み切れる」分量。本作は内省的なモノローグが多く情報密度が高いため、やや短めに設定することで読者の集中力を保つ。

## 各話の構造ルール

- 各話が独立した読み物としても成立するよう、1話の中に小さな「問い」と「進展」を入れる
- 各話の末尾は、次を読みたくなる「引き」で終わる
- 冒頭に2〜3行の短い「前回のふりかえり」を入れる（途中参加の読者向け、くどくならない程度に）

## 公開スケジュール

- **投稿ペース：** 週1回、曜日固定
- **推奨曜日・時間帯：** 火〜木曜日の20〜22時
- **連載期間：** 約3ヶ月（12週）
- **3サイト同時公開：** 同じ曜日・同じ時間帯に3サイトすべてに投稿する

## 公開手順

### 事前準備

1. **カクヨム：** アカウント作成 → プロフィールに作品コンセプト記載 → 作品ページ作成（ジャンル：SF）
2. **Tales：** noteアカウントと連携 → 作品ページ作成
3. **note：** プロフィールに作品コンセプト記載 → 『無響室の対話』専用マガジン作成
4. **共通：** 各サイトのプロフィール・作品概要に「カクヨム／Tales／noteにて同時連載中」と記載
5. **執筆：** 最低3〜4話分を書き溜めてから連載を開始する（理想は全話完成後のストック連載）

### 連載開始時

1. **note：** 「連載開始の告知記事」を投稿（作品の世界観・コンセプト紹介、各プラットフォームへのリンク）
2. **3サイト同時：** 第0話（プロローグ）を公開
3. **X（Twitter）：** 連載開始の告知。作品のテーマに触れる短い文章とリンク

### 毎週の投稿

1. 決めた曜日・時間に3サイト同時公開
2. 各サイトで前話との導線を整備（カクヨムは自動、Tales・noteは手動でリンク追記）
3. X（Twitter）で各話の印象的な一文や問いかけを投稿し、リンクを添える

## noteでの周辺コンテンツ戦略

小説本編とは別に、noteで以下のようなエッセイ記事を不定期に投稿する。これにより「AIと社会」に関心のある読者を小説本編へ誘導する。

- 執筆の裏話（世界観の着想、キャラクターの造形過程など）
- テーマに関する考察（AIが対話を最適化することの功罪、「快適さ」と「摩擦」についてなど）
- 連載の振り返り・次の展開の予告
